# 🎯 Emotion Detection System

## 📌 Project Overview

This project detects human emotions in real-time using a webcam. It uses Deep Learning models (via DeepFace) to analyze facial expressions and display emotions on screen.

---

## ⚙️ Requirements

Make sure the following are installed:

* Python 3.8 or above
* pip (Python package manager)

---

## 📥 Step-by-Step Setup Instructions

### 1️⃣ Clone the Repository

Open terminal / command prompt and run:

```
git clone https://github.com/Gill-Sukham/emotion-detection-system.git
cd emotion-detection-system
```

---

### 2️⃣ Create Virtual Environment (Recommended)

```
python -m venv env
```

Activate it:

**Windows:**

```
env\Scripts\activate
```

---

### 3️⃣ Install Required Libraries

```
pip install deepface opencv-python pyqt5 pyqtgraph
```

---

### 4️⃣ Run the Project

Run any one of the main files:

```
python final_ui.py
```

OR

```
python final_ui_pro.py
```

---

## 📸 Output

* Webcam will open
* Face will be detected
* Emotion (Happy, Sad, Angry, etc.) will be displayed

---

## ⚠️ Notes

* Make sure webcam is connected
* Good lighting improves accuracy
* First run may take time (model loading)

---

## 🧠 Technologies Used
